Windows下的混合编程(C/C++, python, matlab)

2018-08-29
#Windows #C/C++ #Python #Matlab #混合编程

配置环境：Win10 + Visual Studio 2015 + Matlab 2017b + Python 3.5, 其他环境下的配置大同小异, 请根据个人情况自行修改.

环境配置

C/C++调用Python

  1. 下载python并完成python环境配置
  2. 运行Visual Studio,
    • 将***\include添加至VC++包含目录(***为python根目录)
    • 将***\libs添加至VC++库目录
  3. 在链接器->输入->附加依赖项添加”python35.lib”(根据python版本自行修改)
  4. 新建一个cpp文件，输入#include<python.h>，若不报错则配置完成

C/C++调用Matlab

  1. 下载matlab及对应的vs，并完成环境配置
  2. 在matlab命令窗口输入mcc -W cpplib:”name of function” -T link:lib “name of matlab script”(根据情况自行修改””内的内容即可)
  3. 将生成的文件放入同一个文件夹中(可选)，生成的文件有：
    • ***.cpp (***为函数名)
    • ***.def
    • ***.dll
    • ***.exp
    • ***.exports
    • ***.h
    • ***.lib
    • mccExcludedFiles.log
    • readme.txt
    • requireMCRProducts.txt
  4. 将***\extern\include添加至VC++包含目录，并将***\extern\lib\win64\ mingw64添加至VC++库目录
  5. 将生成的.h及.lib文件所在路径(步骤三)分别添加在VC++的包含目录及库目录(或直接将生成的文件加入项目)
  6. 将mclmcr.lib、mclmcrrt.lib、libmex.lib、libmx.lib、libmat.lib、libeng.lib及生成的***.lib文件添加至链接器->输入->附加依赖项
  7. 将相关dll文件拷贝至exe所在路径
  8. 运行demo检测配置是否完成
  • 如果使用的MATLAB是64位的版本，那么C++的程序也要在x64下配置
  • 如果使用的MATLAB是32位的版本，那么C++的程序也要在x86下配置

Python调用C/C++

    python通过DLL文件实现对C/C++的调用，调用方式如下：

import ctypes

lib = ctypes.cdll.LoadLibrary("path of your dll") 
lib."function"()

具体调用方式参考ctypes — A foreign function library for Python

Python调用Matlab

  1. cd至***\extern\engines\python(***为matlab根目录)
  2. 打开cmd，输入python setup.py install
  3. 进入python环境，运行demo检查配置是否完成
import matlab
import matlab.engine

engine = matlab.engine.start_matlab()               # Start MATLAB process
engine = matlab.engine.start_matlab("-desktop")     # Start MATLAB process with graphic UI
engine.sqrt(2.)

Matlab调用C/C++

  1. 打开Matlab完成编译器配置(mex -setup)
  2. 写C++函数，函数的形式必须是
void mexFunction(int nlhs, mxArray *plhs[], int nrhs, const mxArray *prhs[])
    nlhs：输出参数个数
    plhs：输出参数列表
    nrhs：输入参数个数
    prhs：输入参数列表
  1. 将cpp文件拷贝至Matlab当前目录，在Malab命令窗口输入mex “file name”.cpp，即可生成”file name”.mex64/“file name”.mex32文件
  2. 运行demo检测配置是否完成
C++代码：
#include "mex.h"

//MATLAB调用形式： [resizedArr, resizedDims] = ResizeArray(arr, selRows, sekCols)

void mexFunction(int nlhs, mxArray *plhs[], int nrhs, const mxArray *prhs[])
{
    if (nrhs != 3)
    {
        mexErrMsgTxt("参数个数不正确!");
    }

    int iRowNum = mxGetM(prhs[0]);
    int iColNum = mxGetN(prhs[0]);
    double *pArr = (double *)mxGetPr(prhs[0]);

    double *pSelRows = (double *)mxGetPr(prhs[1]);
    double *pSelCols = (double *)mxGetPr(prhs[2]);
    int iRowsRowNum = mxGetM(prhs[1]);
    int iRowsColNum = mxGetN(prhs[1]);

    if (iRowsRowNum != 1 && iRowsColNum != 1)
    {
        mexErrMsgTxt("行参数不正确！");
    }
    int iRowsNum = iRowsRowNum * iRowsColNum;

    int selColsRowNum = mxGetM(prhs[2]);
    int selColsColNum = mxGetN(prhs[2]);

    if (selColsRowNum != 1 && selColsColNum != 1)
    {
        mexErrMsgTxt("列参数不正确！");
    }

    int iColsNum = selColsRowNum * selColsColNum;

    plhs[1] = mxCreateDoubleMatrix(2, 1, mxREAL);
    double *resizedDims = (double *)mxGetPr(plhs[1]);

    resizedDims[0] = iRowsNum;
    resizedDims[1] = iColsNum;

    plhs[0] = mxCreateDoubleMatrix(iRowsNum, iColsNum, mxREAL);
    double *pResizedArr = (double *)mxGetPr(plhs[0]);

//这里因为MATLAB中数据得按列优先
#define ARR(row, col) pArr[(col)*iRowNum + row]
#define RARR(row, col) pResizedArr[(col)*iRowsNum + row]

    for (int r = 0; r < iRowsNum; r++)
    {
        for (int c = 0; c < iColsNum; c++)
        {
            RARR(r, c) = ARR((int)pSelRows[r] - 1, (int)pSelCols[c] - 1);
        }
    }

    mexPrintf("OK!\n");
}
