Win10 + Hexo + github搭建个人博客

2018-08-27
#tutorial

安装步骤

  1. 环境搭建(Node.js, git以及ruby，若配置成功则界面如下：

  1. 完成github账户创建与设置，创建repository并命名为：帐户名.github.io，其他设置如下：

  1. 输入npm install hexo -g完成hexo的安装，安装界面如下

    成功安装后输入hexo -v可得到如下界面：

  2. cd至指定目录(空目录)，输入hexo init，完成该文件夹的初始化

  1. 输入npm install完相关组件的下载

  1. 输入hexo g，启动hexo

  1. 输入hexo s，开启服务器，访问http://localhost:4000/，若页面一直无法跳转，说明端口可能被占用，此时同时按下ctrl+c停止服务器，输入"hexo server -p 端口号”来改变端口号

  1. 进入Git Bash，cd至步骤4所指定目录，如下键入以下命令：

  1. 连续输入以下两个命令： cd ~/.ssh ssh-keygen -t rsa -C “your_github_email” 连续三个回车，生成密钥，得到两个id_rsa和id_rsa.pub两个文件

  1. 输入eval “$(ssh-agent -s)”,添加密钥到ssh-agent；再输入ssh-add ~/.ssh/id_rsa，将生成的ssh key添加到ssh-agent

  1. 登录github，点击头像下方的settings，选择SSH and GPG keys，将~./ssh路径下的id_ras.pub内的key复制到key中
  2. 输入ssh -T git@github.com，测试添加ssh是否成功。如果看到Hi后面是你的用户名，就说明成功了
  3. 在步骤4建立的文件夹中找到_config.yml，按照下图完成配置
  4. 输入npm install hexo-deployer-git –save安装扩展
  5. 输入hexo new post “博客名”，新建一篇博客

  1. 编辑好文章后，输入hexo d -g完成部署
  2. 进入自己的博客主页即可看到刚刚发布的文章
